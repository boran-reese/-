# AI全场景拟态人|实时交互系统|本地私有化部署|云部署|
瑞思智控专注于将AI技术与人性化设计相结合，致力于通过“全场景拟态人”这一创新产品革新传统的人机交互模式。
全场景拟态人是一款高度集成的数字人实时交互系统，它不仅能够实现自然流畅的实时交互，还融合了大型语言模型（LLM）和检索增强生成（RAG）知识库的强大能力，以及声音克隆文本转语音（TTS）和自动语音识别（ASR）技术，应用于各个场景中。
